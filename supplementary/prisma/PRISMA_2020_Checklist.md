## MASEM-Specific Additions to PRISMA

The following elements are specific to meta-analytic structural equation modeling and should be reported:

| Item | Description | Reported in Section |
|------|-------------|---------------------|
| **Pooled Correlation Matrix** | Report the full 12×12 pooled correlation matrix (Stage 1 TSSEM output) with diagonal values, heterogeneity statistics (τ², I²) per cell. | Results: Table 4 |
| **Model Specification** | Clearly specify all three competing SEM models: (1) paths included, (2) constraints (if any), (3) number of free parameters, (4) theoretical justification. | Methods: Model Specification; Figure S1-S3 (path diagrams) |
| **Stage 1 TSSEM** | Report multivariate meta-analysis results: pooled r for each of 66 pairs, 95% CI, I², τ². | Results: Table 3 |
| **Stage 2 TSSEM** | For each model: χ², df, p-value, CFI, TLI, RMSEA (90% CI), SRMR, AIC, BIC. Report all path coefficients (β) with SE and 95% CI. | Results: Table 5-7 |
| **Model Comparison** | Report model comparison statistics (Δχ², ΔAIC, ΔBIC, CFI difference). State preferred model with justification. | Results: Model Comparison; Table 8 |
| **Network Analysis** | Report centrality indices (Strength, Betweenness, Closeness) for each construct. Report community detection results. Present network visualization. | Results: Network Analysis; Figure 7-8 |
| **Bayesian MASEM** | Report prior specifications (Sabherwal et al., 2006 priors vs. weakly informative). Report posterior estimates with 95% credible intervals. Compute Bayes Factors for prior-to-posterior shift. | Results: Bayesian MASEM; Table 9 |
| **Positive-Definite Adjustment** | If correlation matrix required adjustment for non-positive-definiteness, report method (smoothing, ridge adjustment) and magnitude of change. | Methods: Data Preparation; Results footnote |
| **Missing Data Handling** | Report percentage of missing cells in correlation matrices. Describe imputation method (if used) or listwise deletion approach. | Methods: Missing Data |
| **Construct Harmonization** | Report inter-coder agreement statistics for construct mapping (Cohen's κ, percent agreement). Report number of ambiguous cases requiring expert adjudication. | Results: Data Extraction Quality |
